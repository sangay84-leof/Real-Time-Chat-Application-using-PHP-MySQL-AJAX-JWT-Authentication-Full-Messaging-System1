# Chat Frontend

A production‑ready, frontend‑only chat application skeleton.

## Folder Structure

```
chat-frontend/
├─ index.html
├─ assets/
│  ├─ css/
│  │  ├─ style.css
│  │  └─ chat.css
│  ├─ js/
│  │  ├─ app.js
│  │  ├─ queue.js
│  │  ├─ ui.js
│  │  └─ mock-server.js
│  ├─ img/
│  │  └─ icons/   # placeholder for icons
│  └─ fonts/      # optional custom fonts
├─ components/
│  ├─ header.html
│  ├─ chat-box.html
│  └─ footer.html
├─ styles/
│  ├─ variables.css
│  └─ layout.css
├─ scripts/
│  ├─ events.js
│  ├─ renderer.js
│  └─ utils.js
└─ README.md
```

## Development

Open `index.html` in a browser. The mock server will emit fake messages every few seconds.
