# Real-Time-Chat-Application-using-PHP-MySQL-AJAX-JWT-Authentication-Full-Messaging-System1
A real-time chat application built using PHP and MySQL with AJAX for fast message updates. Includes user authentication, responsive UI, secure CRUD operations, and dynamic messaging powered by a circular queue for efficient message handling.
This project is a real-time chat application developed using PHP and MySQL. It implements AJAX to send and receive messages instantly without page reload and features a circular queue data structure for optimized message storage and fast retrieval.

Key Features:
• User registration & login system
• Real-time messaging with AJAX
• Circular queue implementation for message handling
• MySQL database with secure CRUD operations
• Responsive and clean chat UI
• Scalable backend architecture for multiple users

Technologies Used:
PHP • MySQL • AJAX • HTML • CSS • JavaScript
